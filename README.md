# ProperKorean

올바르지 않은 한국어 폰트와 오역을 수정하는 모드입니다.

* 일부 글자가 `?`로 표시되던 폰트 오류 수정
* 폰트 해상도 개선
* 다양한 오역 및 비문 수정, 용어 통일
